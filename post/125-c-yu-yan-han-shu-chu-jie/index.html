1.25-C语言-函数（初阶）

2026-01-25 4 min read

函数

子程序：是一个大型程序中的某部分代码，由一个或多个语句块组成。负责完成某项特定的任务，而且相较于其他代码，具备相对的独立性！
一般会有输入参数并有返回值，提供对过程的分装和细节的隐藏。这些代码通常被集成为软件库。
在一个函数内部复合语句中定义的变量的作用域只能在{}中有效

库函数

库函数一定要用#include<对应的头文件.h>
常见的库函数类型:

  1. IO函数（input/output）
  2. 字符串操作函数
  3. 内存操作函数
  4. 时间/日期函数
  5. 数学函数
  6. 其他库函数

自定义函数

tips :积极尝试阅读英文文档，学会利用英文文档来学习C语言的一些函数，建议养成这个良好的习惯。

实现两个数值互相交换：
void swap(int x, int y)
{
    int z = 0;
    z = x;//借助中间变量来实现交换功能
    x = y;
    y = z;
}
int main()
{
    int a = 0;
    int b = 0;
    printf("交换前:");
    scanf("%d %d", &a, &b);
    swap(a, b);
    printf("交换后:%d %d ", a, b);//发现并没有实现互相交换的效果
    return 0;
}
错误分析:a 与 b是实参，x ，y,z是形参，在主函数与自定义函数中，参数的传递是单向的，实参->形参，它们都有独立的，有自己独一的地址，我们在swap函数中实现了形参的交换，但是并不会影响实参。
当实参传递给形参的时候，形参是实参的一份临时拷贝，对形参的修改不会影响实参!

正确做法:
void swap(int* x, int* y)
{
    int z = *x;
    *x = *y;
    *y = z;
}
int main()
{
    int a = 0;
    int b = 0;
    printf("交换前:");
    scanf("%d %d", &a, &b);
    swap(&a,&b);
    printf("交换后:%d %d ", a, b);
    return 0;
}

函数的参数

实际参数（实参）

真实传给函数的参数，叫实参。
实参可以是：常量、变量、表达式、函数等。
无论实参是何种类型的量，在进行函数调用时，它们都必须有确定的值，以便把这些值传给形参。

形式参数（形参）

形式参数是指函数名后括号中的变量，以为形式参数中只有在函数调用的过程中才实例化（分配内存单元）
，所以叫形式参数，形式参数当函数调用完成之后就自动销毁了，形式参数只有在函数内部有效!
当实参传递给形参的时候，形参是实参的一份临时拷贝!
对形参的修改不能改变实参！

函数调用

传值调用

  • 函数的形参和实参分别占有不同的内存块，对形参的修改不会影响实参!

传址调用

  • 传址调用是把函数外部创建变量的内存地址传给函数参数如的一种调用函数的方式。
  • 这种传参方式可以上函数和函数外边的变量建立起真正的联系，也就是函数内部可以直接操作函数外部的变量。
传址调用：
void swap(int * pa,int *pb)//pa 与 pb 是两个指针变量，作为形参来接收a与b的地址。
{
    int z =*pa;//z = a;
    *pa = *pb;// a = b;
    *pb = z;//b = z;
    //通过解引用操作，间接访问外部参数!
}
int main()
{
    int a = 5;
    int b = 10;
    swap(&a,&b);//分别取出a与b的地址，传给函数swap
    return 0;
}
