2.21-C二级指针（初阶）

2026-02-21 2 min read

二级指针

二级指针变量是用来存放一级指针变量的地址的！

int a = 10;
int* pa = &a;
int** ppa = &pa;
*(*ppa) = 20;//双重解引用操作
//*pa = 20;
printf("%d\n",a);

指针数组

指针数组是数组！
用来存放指针的数组是指针数组。

用指针数组模拟二维数组
int main()
{
	int arr1[4] = { 1,2,3,4 };
	int arr2[4] = { 2,3,4,5 };
	int arr3[4] = { 3,4,5,6 };
	int* parr[3] = { arr1,arr2,arr3 };
	int i = 0;
	for (i = 0; i < 3; i++)//行
	{
		int j = 0;
		for (j = 0; j < 4; j++)//列
		{
			printf("%d ",parr[i][j]);//parr[i][j]并不代表parr是一个二维数组，再次验证了[]只是一个运算符而已，自带解引用功能，arr[i]<=>*(arr+i)
            //parr[0][j]==arr1[0]<=>*(arr1+0)
            //parr[i][j]   --->  *(parr[i]+j)  "parr[i]"可以看成指针数组的一个元素
		}
		printf("\n");
	}
	return 0;
}//parro == arr1
//arr1[0]
