转义字符

转义字符：是一种具备特殊功能且屏幕上不可见的字符，以字符''开头。

重要的转义字符：
1. \' 输出字符' 用法printf('\''');
2. \"
3. \? 在书写连续多个问好时被使用，防止他们被解析成三字母词
4. \\   用于表示一个反斜杠，防止它被解释为一个转义序列符 用法：printf("c:\\test\\test.c");
5. \n 换行符，将光标当前位置移到下一行开头
6. \r  回车，将光标当前位置移到本行开头
7.  \ddd ddd表示1~3个八进制的数字。如：\130 (八进制数字范围是0~9)
8. \xdd 此时dd表示2个十六进制的数字。如：\x30 
9. \t 水平制表符，一个字符值，效果相当于从键盘输入一个TAB键。
不太重要的转义字符：
1. \a 发出警告声
2. \b 退格
3. \f 换页
4. \v 垂直

#include<stdio.h>
{
    printf("%d\n",strlen("abcdef"));//6
    printf("%d\n",strlen("c:\test\628\test.c"));//14
    return 0;
}

注释

注释是给人看的，对编译器没什么影响。
总之，要养成写注释的好习惯，不仅可以梳理写代码时的逻辑思路，还能够加深对代码的印象，同时也方便以后自己复盘检查。

C语言注释风格/*代码*/不支持嵌套注释
C++注释风格//代码
