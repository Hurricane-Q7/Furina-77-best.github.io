1.21-C语言分支、循环语句（初阶）

2026-01-21 3 min read

for 循环

for循环是一种更为方便的循环语句,比while更加灵活!
for(exp1;exp2;exp3)
{
循环语句；
}

  • exp1:循环变量初始化，仅仅执行一次。
  • exp2:循环变量判断，用来判断是否继续循环，执行循环体前先执行这个，一般为一个条件表达式（值为 0与!0）,结果为逻辑真执行，反之不执行。
  • exp3:循环的调整。
    执行顺序 ：先执行表达式1，再执行表达式2，看满足不满足条件，满足条件执行循环体，否则退出循环，执行完循环体后，再执行表达式3，让后再执行表达式2！
    break只能结束一个循环,双重循环中内层循环中有一个break,那么这个break只能中断内层循环!
    tips:
  • 不可在for循环体内修改循环变量，防止for循环失去控制!

for循环的变种

for循环里面的表达式是可以省略的，但是不要轻易省略！

for(;;;)
{语句;}死循环！
for中的三个表达式都可以省略。
对比下面两个代码的区别:
第一个
int i=0;
int j=0;
for(i=0;i<10;i++)
{
    for(j=0;j<10;j++)
    {
        printf("hehe\n");//内层执行10次后j会再次初始化为0
    }
}
第二个:
只省略两个for循环的变量初始化部分!
int i=0;
int j=0;
for(;i<10;i++)
{
    for(;j<10;j++)
    {
        printf("hehe\n;");//这个代码就不会再次初始化!
    }
}
for循环也可以多变量来使用:
int x, y;
for (x = 0, y = 0; x < 2 && y < 5; ++x, y++)
{
    printf("hehe\n");
}

一道笔试题:执行了几次循环!

#include<stdio.h>
int main()
{
    int i = 0;
    int k = 0;
    for(i=0,k=0;k=0;i++)
    k++;
    return 0;
}//0次循环!

条件判断阶段：执行k=0—— 这是赋值操作，不是判断操作！
赋值语句k=0的返回值就是被赋的值（也就是0）；
C 语言中0表示 “假”，非 0 表示 “真”，因此条件表达式直接返回 “假”；
循环退出：条件为假，循环体k++一次都不会执行，直接退出for循环。

do while循环

格式:
do
{
循环体;
}while(条件判断);
特点:至少循环一次!先循环一次，再判断！
