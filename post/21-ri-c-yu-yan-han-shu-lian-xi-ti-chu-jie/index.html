求任意两个数的最大公约数

约数

约数：如果一个整数 a 除以另一个整数 b（b=0），商是整数且没有余数，就说 b 是 a 的约数，a 是 b 的倍数。
a/b:可以叫a除以b,a是被除数，b是除数。也可以叫做b除a!
✅ 核心特点：约数是能整除某个数的整数，一个数的约数数量是有限的，最小约数是 1，最大约数是它本身。

最大公约数

定义：几个整数共同拥有的约数，叫做这几个数的公约数；其中最大的那个公约数，就是这几个数的最大公约数。
✅ 核心特点：公约数一定包含 1（所有整数都能被 1 整除），两个数的最大公约数不会大于其中较小的那个数。

辗转相除法（求最大公约数）

int main()
{
	int a = 0;
	int b = 0;
	int c = 0;
	printf("输入：");
	scanf("%d %d", &a, &b);
	while (c=a % b)
	{//c=a%b,如果c等于0，那么b就是二者的最大公约数
    //c!=0辗转一下，除数边被除数，余数变除数
		a = b;//
		b = c;
	}
	printf("最大公约数是：%d\n", b);
	return 0;
}

编写一个程序统计1~100数中的整数一共包含了多少个数字九

//个位：9 19 29 39 49 59 69 79 89 99
//十位：90 91 92 93 94 95 96 97 98 99
int main()
{
	int count = 0;
	int i = 1;
	for (i = 1; i <= 100; i++)
	{
		//判断个位
		if (i % 10 == 9)//余数=被除数- 除数*商
		{
			count++;
		}
        //判断十位
		if (i / 10 == 9)//i/10,是拿掉一个数个位的写法，因为两边执行的都是整数的除法运算
		{
			count++;
		}
	}
	printf("count = %d", count);
	return 0;
}

求和

int main()
{
	int i = 1;
	double sum = 0;
	int flag = 1;
	for (i = 1; i <= 100; i++)
	{
		sum = sum + flag * (1.0 / i);
		flag = -flag;//1 -1 1 -1循环交替
	}
	printf("%lf", sum);
	return 0;
}
