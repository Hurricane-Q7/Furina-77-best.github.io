1.17日-C语言指针与结构体（初识）

2026-01-17 3 min read

内存

内存 是电脑上特别重要的存储器，计算机中程序的运行都是在内存中进行的。
内存会划分为一个个的内存单元，每个内存单元的大小是1byte，为了能够有效访问到内存的每个单元，就给内存单元进行了编号，这些编号被称为该内存单元的地址。
belike:这些编号类似于每个房间的门牌号，方便查找。

int main()
{
    int a=10;//向内存申请4byte的内存空间。
    &a;
    printf("%p\n",&a);//以十六进制的表现形式打印a的地址,内存单元中第一个字节单元的地址。
    return 0;
}

指针

C语言中指针就是地址，地址就是指针。

指针变量

指针变量就是用来存储地址（指针）的一个变量而已。

int main()
{
    int a=10;
    int* p=&a;//'&'是一个取地址操作符，取出a的地址放到p里面。'*'这里表示变量p是一个指针变量，int说明p指向的int类型的数据。
    *p=20;//*是一个解引用操作符，通过p里面的存放的地址来找到该地址指向的对象。
    printf("%d\n",a);
    return 0;
}

指针变量的大小

指针变量的大小取决于一个地址存放的时候需要多大的空间，取决于系统是32位还是64位。
32位机器上的地址：32bit位-4byte，所以指针变量的大小是4byte。-->32个0与1组成的地址长度。
64位机器上的地址：64bit位-8byte，所以指针变量的大小是8byte。-->64个0与1组成的地址长度。

结构体

C语言允许用户自己建立由不同的类型数据组成的组合型数据结构，它被称为结构体。
表现形式:
struct 结构体名字
{成员表列}；//成员表列也称为域表。
成员运算符：

  1. .运算符
  2. ->运算符
    C语言的结构体可以用来描述复杂的数据对象。
    C语言就给了自定义类型的能力，自定义类型有一种叫:结构体（struct）
struct Stu//结构体也遵循先定义在使用的规则
{
    char name[20];
    int age;
    char sex[10];
    char tele[12];
};
void print(struct Stu* ps)
{
    printf("%s %d %s %s\n", (*ps).name, (*ps).age, (*ps).sex, (*ps).tele);
    printf("%s %d %s %s\n", ps->name, ps->age, ps->sex, ps->tele);
}
int main()
{
    struct Stu s = {"zhangsan",20,"nan","15596668862"};//创建结构体变量s,对他进行初始化。
    printf("%s %d %s %s\n",s.name,s.age,s.sex,s.tele);//结构体对象.成员名
    print(&s);
    return 0;
}
