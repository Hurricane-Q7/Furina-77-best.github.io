逻辑操作符

逻辑与 逻辑或

逻辑或（||）：一真即真，全假才假。
逻辑与（&&）：一假即假，全真才真。
逻辑或就是或者，逻辑与就是并且的意思！

int main()
{
	int i = 0, a = 0, b = 2, c = 3, d = 4;
	i = a++ && ++b && d++;
	printf("a = %d\nb = %d\nc = %d\nd = %d\n", a, b, c, d);
	return 0;
}
//运算规则：
//&& 左边为假，右边就不计算了
//  ||  左边为真，右边就不计算了
//只要逻辑操作符两边的一个式子能够得出整个式子结果就不计算另一个式子。

#条件操作符（三目操作符）
exp1?exp2:exp3

逗号表达式

逗号表达式，就是用逗号隔开的多个表达式。
逗号表达式，从左向右依次执行。整个表达式的结果是最后一个表达式的结果。
(exp1,exp2,exp3);

//代码一
int a = 1;
int b = 2;
int c = (a>b, a=b+10,a,b=a+1);
//代码二
a = get_val();
count_val(a);
while(a>0)
{
    a = get_val();
    count_val(a);
}
改写为逗号表达式
while(a = get_val()，count_val(a)，a>0)
{
    a = get_val();
    count_val(a);
}

下标引用、函数调用与结构体成员

下标引用

[]

arr[7]->*(arr+7)->*(7+arr)->7[arr]
[]就是一个下标引用操作符，当指向单个数组元素的时候，可以写成上述式子，初始化的时候不可以。

结构体成员操作符

. ->
用法：

  1. 结构体类型变量名.结构体类型成员名字
struct stu
{//定义stu结构体类型名，仅仅是定义一种类型名字，没有变量，不可以在里面对成员进行初始化
	char name[20];
	int age;
	float sint main();
    };
    int main()
{
	struct stu s = { "lichaoxiang",18,100.0 };//创建struct stu 型变量
	printf("%s %d,%.2f", s.name, s.age, s.score);
	return 0;
};
  1. 结构体指针->成员名字
struct stu
{
	char name[20];
	int age;
	float score;
};
int main()
{
	struct stu s = { "lichaoxiang",20,100.0 };//创建stu型变量s并完成初始化
	struct stu* ps = &s;
	printf("%s %d %lf", ps->name, ps->age, ps->score);
	return 0;
}
ps->age完全等价于
(*ps).age
