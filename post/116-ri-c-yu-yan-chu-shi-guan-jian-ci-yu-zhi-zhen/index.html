1.16日-C语言-关键词与宏（初识）

2026-01-16 3 min read

关键词

typedef

typedef 顾名思义就是对类型定义，也就是对类型重命名，再起个名字。

typedef unsigned int uint;//定义完成后用uint再定义变量时和用unsigned int定义是一样的
int main()
{
    unsigned int num1=0;
    uint_32 num2=0;
    return 0;
}

static(静止的、不变的)

C语言中用来修饰变量和函数的

  1. 修饰局部变量-称为静态局部变量
  2. 修饰全局变量-称为静态全局变量
  3. 修饰函数-称为静态函数
    内存中存储变量时有三块空间，栈区、堆区、静态区，static修饰局部变量时将变量从栈区挪到静态区
void test()
{
	static int a = 1;//static修饰局部变量时，局部变量出了作用域，是不会销毁的。本质上，static修饰局部变量时，改变了变量的存储位置的。影响局部变量的生命周期，生命周期变长，和程序的生命周期一样，程序结束，这个变量占用的空间才销毁（还给操作系统）。
	a++;
	printf("%d ", a);
}
int main()
{
	int i = 0;
	while (i < 10)
	{
		test();
		i++;
	}
	return 0;
}运行结果是：2~11

全局变量（大括号外部）具有链接性，举个例子，我在111.c文件中定义了一个全局变量，如果我想在222.c中引用它，要用extern进行链接声明。
static 修饰全局变量的时候，全局变量量的外部链接属性就变成了内部链接属性。其他源文件就不能再使用这个全局变量了。被static修饰的全局变量作用域只变成当前.c的源文件。
函数：也具有链接属性。一个函数本来是具有外部链接属性的，但是被static修饰的时候，外部链接属性就变成了内部链接属性，其他源文件（.c）就无法使用了。
register

int main()
{
    register int num = 3;//建议：3存放在寄存器中，寄存器是一种非常访问速度非常快的存储设备。
    return 0;
}

宏
宏名+宏的参数+宏体

#define Add(x,y) ((x)+(y))
int main()
{
    int a=10;
    int b=20;
    int c=Add(a,b);
    printf("%d\n",c);
    return 0;
}
