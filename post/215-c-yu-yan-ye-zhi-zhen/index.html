2026-02-15 2 min read

野指针

概念 ：野指针就是指针指向的位置是不可知的(随机的、不正确的、没有明确限制的)

  1. 没有初始化，不知道指向谁；
  2. 指针越界访问
  3. 指针指向的空间释放
#include<stdio.h>
int* test()
{
	int a = 10;//局部变量a，作用域在函数每次调用完后，销毁内存空间使用权限，还给OS,但是该空间在内存中还是存在的！
	return &a;
}
int main()
{
	int* p = test();//定义p来接收test的返回值（局部变量的地址），调用完后该地址失效，所以p是野指针！
    //printf("haha\n");
	printf("%d", *p);//打印结果是10，但是不代表永远是10,可能被上一次函数栈帧覆盖！
	return 0;
}

避免野指针的一些coding habits

1. 指针初始化
//int * p = NULL;
if(p!=NULL)
{
    *P = 10;
}
2. 小心指针越界
3. 指针指向空间释放即使变成NULL
4. 避免返回局部变量的地址
5. 指针使用之前检查有效性
