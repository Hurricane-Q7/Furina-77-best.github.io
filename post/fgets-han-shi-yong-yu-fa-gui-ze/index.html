fgets函使用语法规则

2025-12-28 2 min read

fgets函数

代码块

char *fgets(char *str,int n,FILE *stream);
str:指向字符数组的指针（地址），用于存储读取的字符串。
n:要读取的最大字符数，实际读取n-1个字符（包括空字符\0）;
stream: 文件流，通常使用stdin表示标准输入。

#include<stdio.h>
int main(){
    char str[100];
    printf("Enter a string:");
    fgets(str,sizeof(str),stdin);
    printf("You enntered: %s",str);
    return 0;
}

返回值

返回值：
成功：返回 str 的首地址；
失败 / 读到文件末尾（EOF）：返回 NULL；
若读取过程中遇到换行符 \n，会将 \n 一并读取到 str 中，再补 \0

核心语法规则

① 读取终止条件（满足其一即停止）：
读取了 num-1 个字符；
读到换行符 \n（\n 会被保留在字符串中）；
读到文件末尾（EOF）。
② 内存安全：强制最多读取 num-1 个字符，剩余 1 个位置留给 '\0'，彻底避免 gets 的缓冲区溢出问题（gets 无长度限制）。
③ 参数合法性：
num 必须 ≥ 1（否则直接返回 NULL）；
str 必须是有效的内存地址（不能是 NULL，也不能是未分配的野指针）。
