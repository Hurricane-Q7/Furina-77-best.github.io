链式访问，依赖函数的返回值

链式访问就是一个函数的返回值，当作另一个函数的参数
函数不写返回值的时候，默认返回类型是int型

int main()
{
	char arr[20] = "hello";
	int ret = strlen(strcat(arr, "bit"));//链接两个字符串的后面都有'\0',连接时将两个字符串1后面的'\0'取消，只在新的字符串保留'\0'。
	printf("%d\n", ret);//打印结果是8,strlen只统计'\0'之前的字符个数!
	return 0;
}

函数的声明与定义

函数的声明

  1. 告诉编译器函数的名字、参数、返回类型，但是具体函数是不是存在的，函数声明决定不了
  2. 函数的声明一般出现在函数的使用之前，要满足先声明后使用。
  3. 函数的声明一般放在头文件当中，在主函数中想要调用该函数，只需要在调用前先包含该函数的头文件即可，函数的头文件里面一般存放的是函数的声明部分，函数的定义部分放在别的源文件里面即可。

函数的定义

函数的定义是指函数的具体实现，交待函数功能的实现。

int add(int x,int y);//函数声明
int main()
{

}
int add(int x,int y)//函数定义
{
    return x+y;
}
