猜数字小游戏

时间戳;时间戳（Timestamp）是一种用于唯一标识某个时间点的数字或字符串标记，它的核心作用是把人类可读的日期时间转换成计算机易于存储、计算和比较的格式。
一句话:时间戳就是把 “生活时间” 翻译成 “计算机能看懂的数字时间”，简单、统一、好用。
时间戳的起点:1970年1月1日 00:00:00 UTC起点（Unix 纪元，计算机界的规定），从那一刻世时间戳一直在增加。

rand与srand函数

rand生成一个伪随机数。
rand(void)不需要参数。
rand是包含于<stdlib.h>中的一个用来生成随机数的一个库函数，返回值是整数（0~RAND_MAX(32767))。
注意:使用rand函数前要先用srand函数函数来为伪随机数生成器设置随机数种子，否则结果可能不尽人意。

srand

srand:Sets a random starting point。设置一个起点.例如:srand((unsigned int)time(NULL));设置一个时间戳起点.
void srand(unsigned int seed);//无符号整型的种子;

srand(100);
int ret = rand();
printf("%d",ret);//反复调试，发现结果不变，没有达到随机的效果，原因是srand中的时间种子一直不变。需要时时刻刻都在发生变化的东西，也就是时间!

time函数

time函数用来获取系统时间，返回时间戳
time_t time(time_t *timer);time_t本质上就是一个整形数据类型，用typedef下的定义.
NULL就是空指针，指针领域里面的0,在vs 2022 中对转到定义发现,NULL（（void *）0）。

goto语句

goto语句是一种控制语句，适用于跳出多级循环，使用要慎重慎重再慎重！

 #include<stdio.h>
#include<string.h>
#include<stdlib.h>
int main()
{
	again:
	system("shutdown -s -t 60");
	printf("你的电脑将在60秒后关机,赶紧输入我是猪来停止关机!\n");
	char input[20]={0};
	scanf("%s",input);
	if(strcmp(input,"我是猪")==0)
	{
		system("shutdown -a");
		printf("你的电脑已关机，哈哈哈!\n");
	}
	else 
	{
		goto again;//这里的goto语句，也可以用循环来代替。
	}
	return 0;
}
