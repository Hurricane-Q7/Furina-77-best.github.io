1.15号-C语言-操作符(初识)

2026-01-15 3 min read

操作符

C语言中有很多操作符

算术操作符
+ - * /（除法操作符） %（求余数）++ --
int a = 7/2;//此时a=3,也就是商，当除号两端都是整数时，执行的是整数除法，如果两端只要有一个浮点数或两个浮点数就执行浮点数除法。
 例如：
 float a = 7 / 2.0;//
 printf("%.2f",a);//结果为3.50
 取模操作符的两个数只能是整数
 int b = 7%2;//b == 1;

***位操作符***
移位操作符：
<<左移  右移>>
位操作符:
& ^ |

赋值运算符
1. =；a = 3;
2. +=;a = a+3等价于a+=3;+号在前面，先执行+号，后面的不言而喻了。
3.-=；
4.*=；
5. /=；
&=;^=;|=;>>= <<=;

单目操作符（只有一个操作对象）
1. !逻辑反操作，真的变假，假的变真
2. + -
3. &取地址操作符，单个操作对象为取地址，双个对象为按位与操作符！
4. sizeof,这是一个单目操作符，并不是一个函数，常用来求操作数的类型长度，单位是Byte
常见用法:
求数组元素个数:
int arr[10];
int len = sizeof(数组名)/sizeof(数组的一元素);
printf("%d\n",sizeof(arr));//计算的是整个数组的大小，共40BYTE
其次,
sizeof(int);//计算类型的长度时，括号不可省略。
5. ~ 对一个数的二进制按位取反
6. -- ++
7.* 间接访问操作符（指针的解引用操作）
8.（类型）强制转换数据类型
int a = (int)3.14;//字面浮点数3.14，编译器默认理解位double 类型
printf("%d\n",a);

关系操作符
>  
>= 
< 
v
<= 
!= 不相等
==相等
优先级：
< > <= >=这四种相同优先级，按右结合性处理。
== 与！= 优先级最低且相同。
优先级: 算术运算符>关系运算符>赋值运算符

逻辑操作符
&& 逻辑与（and）一假即假 全真才真
|| 逻辑或(or) 一真即真

条件表达式（三目表达式）
(exp1?exp2:exp3)
exp1为真，则exp2作为整个条件表达式的值
exp1为假，则exp3作为整个条件表达式的值
int a=10,b=20;
int r = (a>b?a:b);//r==20
return 0;

逗号表达式：用逗号隔开的一串表达式；
语法规则：从左向右依次计算,整个表达式的结果是最后一个表达式的结果
