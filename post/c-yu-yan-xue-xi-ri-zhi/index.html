2025-12-29 2 min read # C语言学习笔记

1.1

计算机语言：人和计算机交流用的语言！
C/C++是一种编译型语言（编译-->链接-->可执行程序.exe）
软件分上层与底层软件：
1.上层软件：操作系统上层的软件，比如应用软件（qq,微信等等）
2.底层软件：操作系统及以下的软件层序，而C语言擅长开发底层软件。

计算机语言发展：低级--->高级

二进制语言-->汇编指令（助记符）-->B语言-->C语言

C标准

ANSI C
ANSI C是由美国国家标准协会（ANSI）制定的C语言标准
eg:C89 C90 C99 C11
常见编译器：Clang、GCC、WIN-TC、MSVC(编辑器，vs底层)、Turbor C
