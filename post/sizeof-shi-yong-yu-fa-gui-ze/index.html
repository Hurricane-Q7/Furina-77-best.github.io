sizeof使用语法规则

2025-12-28 2 min read

sizeof

sizeof 是 C 语言的单目运算符（并非函数），核心作用是计算 “数据类型 / 变量 / 表达式” 在内存中占用的字节数，编译器就会确定结果（不执行表达式），是内存管理、类型适配的核心工具。

常见表现形式

  1. 计算变量占用的字节数
    sizeof(变量名); // 括号可省略（变量名时），如 sizeof var
  2. 计算数据类型占用的字节数
    sizeof(数据类型); // 括号不可省略，如 sizeof(int)
  3. 计算表达式结果的类型占用的字节数（表达式不执行）
    sizeof(表达式); // 如 sizeof(a + b)

计算数组总字节数（关键）

sizeof(数组名) 能直接获取数组在内存中占用的总字节数，结合单个元素的字节数，可计算数组长度：

char str[100];
int arr[5] = {1,2,3,4,5};

// 数组总字节数
printf("str总字节数：%zu\n", sizeof(str));  // 输出：100（100个char，每个1字节）
printf("arr总字节数：%zu\n", sizeof(arr));  // 输出：20（5个int，每个4字节）

// 计算数组长度（元素个数）
int arr_len = sizeof(arr) / sizeof(arr[0]); 
printf("arr元素个数：%d\n", arr_len);       // 输出：5
⚠️ 易错点：数组名作为函数参数时，会退化为指针，此时 sizeof(数组名) 取的是指针的字节数（32 位系统 4 字节，64 位 8 字节），而非数组总字节数.
