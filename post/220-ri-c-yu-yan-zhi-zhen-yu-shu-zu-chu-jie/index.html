C语言指针与数组

指针-指针

意义：指针减去指针的结果的绝对值代表两个两个指针对应的元素个数！
注意：指向同一块空间的2个指针才能相减！（比如指向同一个数组）！

int my_strlen(char* str)
{
	char* start = str;//记住初始地址
	while (*str != '\0')//只统计\0之前的字符个数核心代码
	{
		str++;
	}
	return str - start;//返回str与start中间有几个数字！
}
int main()
{
	int len = my_strlen("abcde");
	printf("%d", len);
	return 0;
}

指针的关系运算（比大小）

标准规定:
允许指向数组的指针与指向数组最后一个元素后面的那个内存位置的指针比较，但是不允许与允许指向数组元素的指针与指向数组最后一个指向第一个元素之前的那个内存位置的指针进行比较。
p3可以与p2比较，与p1不可以

#define N_VALUES 5
int main()
{
	
	float value[N_VALUES];
	float* vp;
	for (vp = &value[N_VALUES]; vp > &value[0];)
	{
		*(--vp) = 0;
	}
	return 0;
}

指针与数组

一句话：一般来说数组名是数组首元素的地址，通过这个指针可以间接访问数组的各个元素！
补充：

  1. sizeof(数组名)，计算整个数组的大小，sizeof内部单独放一个数组名，该数组名表示整个数
    组。 printf("%d \n",sizeof(arr));//40
  2. &数组名，取出的是数组的地址。&数组名，该数组名表示整个数组。
    除此1,2两种情况之外，所有的数组名都表示数组首元素的地址。
void test(int* p, int sz)
{
	int i = 0;
	for (i = 0; i < sz; i++)
	{
		printf("%d ", *(p + i));
	}
}
int main()
{
	int arr[10] = { 0 };
	int sz = sizeof(arr) / sizeof(arr[0]);
	test(arr, sz);//将首地址与元素个数传过去
	return 0;
}
