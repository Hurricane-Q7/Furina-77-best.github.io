2026-02-22 2 min read

结构体

结构是一些值的集合，这些值称为成员变量。结构的每个成员可以是不同类型得变量！
总结
结构体是用户自定义的复合数据类型，而非 C 语言内置类型。
核心作用是将不同类型的变量打包成一个整体，描述复杂对象。

结构体声明部分

struct peo//声明struct peo 数据类型
{
    char name[20];
    char tele[12];//成员列表
    char sex[5];
    int high;
}变量列表;//声明一个struct peo的数据类型，不能在结构体类型声明环节内部
//初始化，该部分不占用内存空间！
int main()
{
    struct peo p1;//创建struct peo类型的变量，拿着图纸盖房子，占用内存空间！
    return 0;
}

结构体成员

结构的成员可以是标量、数组、指针、甚至是结构体变量！

struct stu
{
……;
};
struct peo
{
  struct stu s;
  int num;
  int score;
};

结构体成员访问

主要通过两个操作符

  1. 结构体变量.member
  2. 结构体变量指针->member
